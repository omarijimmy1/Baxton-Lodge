# Vercel Deployment

Your site is now configured for deployment with Vercel!

## Setup Instructions:

1. **Connect your GitHub repository to Vercel:**
   - Go to https://vercel.com/import
   - Select "GitHub" and authorize
   - Find and import `omarijimmy073-sketch/Baxton-Lodge`

2. **Add environment variables to Vercel:**
   In your Vercel project settings, add:
   - `VERCEL_TOKEN` (from Vercel account settings)
   - `VERCEL_ORG_ID` (from Vercel team/account)
   - `VERCEL_PROJECT_ID` (shown after import)

3. **Deploy:**
   - Your site will auto-deploy on every push to `main`
   - Access your site at: `https://baxton-lodge.vercel.app`

**Note:** The GitHub Pages workflow (`deploy.yml`) is no longer needed and can be deleted.
